Types of Solutions

Contextual advertising is an automated process where a promotional message is matched to relevant digital content. The algorithms which underpin contextual advertising, select the advertisements based on keywords and other metadata included in the content. This is not an identity technology per se, but rather an identity-less alternative for programmatic targeting.

Unlike third-party data that is often available to many different companies, first-party data is unique to a publisher/advertiser. It's data that a publisher owns and collects with direct consent from consumers, so it is limited to the consumer actions within a single domain or a small group of domains owned by the same publisher.

First party

A data clean room is a piece of software that enables advertisers and brands to match user-level data without actually sharing any PII/raw data with one another. Major advertising platforms use data clean rooms to provide advertisers with matched data on the performance of their ads on t heir platforms. Alternative third-party vendors also look to be intermediaries to advertisers and publishers to match data and target across the wider network.

Clean Rooms

Cohort-based advertising targets anonymised groups of people who share common interests and online browser behavior. Cohorting may take place with both known, personally identifiable information (PII)-based data assets, and anonymous data assets, like contextual signals found on web pages. Cohort-based advertising is able to identify the individual deterministically across domains, but applies an anonymisation layer to protect user privacy.

Hashed Emails

The Universal ID is a user identifier created by an ad-tech company to provide a shared identity to identify the user across the programmatic ecosystem without the need to sync cookies. Unlike cookies that are based on probabilistic matching, most Universal IDs are created on the basis of deterministic matching. First-party data (CRM) and offline data can be used to create universal IDs, but the most popular signal used to create universal IDs are encrypted email addresses from logged in users..
